# Test

To run the code, serve the `public` directory with a web server and navigate to <http://127.0.0.1:8000>.

With Python:

```bash
python -m http.server -d public -b 127.0.0.1 8000
```

With PHP:

```bash
php -S 127.0.0.1:8000 -t public
```
